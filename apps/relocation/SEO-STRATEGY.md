```
│   ├── /costa-rica
│   └── /mexico
├── /expat-guide (NEW - pillar)

```

### On-Page SEO Checklist
- [ ] Unique title tags (50-60 chars) with primary keyword
- [ ] Meta descriptions (150-160 chars) with call-to-action
- [ ] H1 tags matching primary keyword intent
- [ ] Internal linking between cluster pages
- [ ] Schema markup (FAQ, HowTo, Article)
- [ ] Image optimization with alt text
- [ ] Mobile-first responsive design
- [ ] Core Web Vitals optimization

### Internal Linking Strategy
1. **Hub-and-spoke model:** Link pillar pages to all cluster pages
2. **Cross-cluster links:** Connect related topics (e.g., /guides/spain ↔ /digital-nomad-visa/spain)
3. **Contextual links:** Link from articles to relevant static pages
4. **Navigation:** Add dropdown menus for main topic clusters

## Content Templates

### Country Guide Template
Each country guide should include:
1. **Overview** - Why move to [Country]
2. **Visa Options** - Work, digital nomad, retirement visas
3. **Cost of Living** - Monthly budget breakdown
4. **Best Cities** - Top locations for expats
5. **Healthcare** - System overview and insurance
6. **Banking** - Opening accounts as a foreigner
7. **Language** - Requirements and learning resources
8. **Taxes** - Tax implications for expats
9. **Community** - Expat groups and resources
10. **FAQ** - Common questions (structured data opportunity)
11. **AI Assistant CTA** - Link to /chat and /voice

### Digital Nomad Visa Template
Each visa guide should include:
1. **Visa Overview** - Duration, renewability
2. **Requirements** - Income, insurance, documents
3. **Application Process** - Step-by-step
4. **Costs** - Application fees, processing times
5. **Tax Implications** - Tax residency rules
6. **Pros and Cons** - Honest assessment
7. **Comparison Table** - vs other countries
8. **FAQ** - Structured data
9. **AI Assistant CTA**
