## Optimization Formulas

### Calculator Apps
```
Free [Location] [Type] [Year]. Calculate [Benefit]. [CTA].
```
Example: "Free UK salary calculator 2025. Calculate take home pay instantly."

### Insurance Apps
```
Compare [Type] quotes instantly. Get [Benefit]. [CTA].
```
Example: "Compare drone insurance quotes instantly. Get UK cover. Save today."

### Job Boards
```
Find [Job Type] in [Location]. [Benefit]. [CTA].
```
Example: "Find fractional CFO jobs in UK. Part-time roles. Apply today."

### Services
```
[Action] [Service] for [Benefit]. [Features]. [CTA].
```
Example: "Plan events with UK platform. Find venues. Get quotes."

## Common Fixes Applied

| Issue | Solution | Example |
|-------|----------|---------|
| Too long | Remove redundancy | "Calculate costs for nurseries, childminders..." → "Calculate nursery, childminder..." |
| Keyword stuffing | Use once naturally | "tractor insurance...tractor insurance" → "tractor insurance...farm cover" |
| No CTA | Add strong CTA | Added "Save today" / "Get quotes" |
| Generic | Be specific | "mobility equipment" → "road & pavement use" |
| Verbose | Smart abbreviations | "England, Wales..." → "all UK regions" |

## Adding New Apps

1. Edit `optimize-meta-descriptions.js`
2. Add to `OPTIMIZED_DESCRIPTIONS`:
```javascript
'new-app': 'Description 120-160 chars. Keywords & CTA included.',
```
3. Run: `node optimize-meta-descriptions.js --dry-run`
4. Verify output
5. Apply: `node optimize-meta-descriptions.js`
